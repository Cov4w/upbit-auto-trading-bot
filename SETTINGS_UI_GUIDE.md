### 2. 개별 설정 슬라이더

#### 💰 Trade Amount (매수 금액)
```
💰 Trade Amount          10,000 KRW
├────────●──────────────────────┤
5K                              100K
```
- **범위**: 5,000 ~ 100,000 KRW
- **단위**: 1,000 KRW
- **설명**: 한 번에 매수할 금액

#### 🎯 Target Profit (목표 수익률)
```
🎯 Target Profit              2.0%
├────────●──────────────────────┤
0.5%                            20%
```
- **범위**: 0.5% ~ 20%
- **단위**: 0.1%
- **설명**: 이 수익률 달성 시 자동 매도
- **색상**: 녹색

#### 🛑 Stop Loss (손절률)
```
🛑 Stop Loss                  2.0%
├────────●──────────────────────┤
0.3%                            20%
```
- **범위**: 0.3% ~ 20%
- **단위**: 0.1%
- **설명**: 이 손실률 도달 시 자동 매도
- **색상**: 빨간색

#### 🔄 Rebuy Threshold (재매수 하락폭)
```
🔄 Rebuy Threshold            1.5%
├────────●──────────────────────┤
0%                              10%

After profit, rebuy only if price drops by this amount
```
- **범위**: 0% ~ 10%
- **단위**: 0.1%
- **설명**: 익절 후 가격이 이만큼 하락해야 재매수 허용

## 🔧 사용 방법

### 방법 1: 프리셋 사용 (빠름)
1. 원하는 전략 버튼 클릭 (⚡ Scalping / 🛡️ Swing / 🚀 Bull)
2. 슬라이더 값이 자동으로 변경됨
3. "✅ Apply Settings" 버튼 클릭
4. 완료!

### 방법 2: 수동 조절 (세밀함)
1. 각 슬라이더를 드래그하여 원하는 값 설정
2. 실시간으로 값 확인
3. "✅ Apply Settings" 버튼 클릭
4. 완료!

## 🎯 설정별 효과

### Trade Amount (매수 금액)
- **높게 설정**: 수익도 크지만 리스크도 큼
- **낮게 설정**: 안전하지만 수익도 적음
- **추천**: 계좌의 10-30% 정도

### Target Profit (목표 수익률)
- **높게 설정**: 큰 수익 노리지만 기회가 적음
- **낮게 설정**: 자주 매도되지만 작은 수익
- **추천**: 변동성에 따라 1-5%

### Stop Loss (손절률)
- **높게 설정**: 손실 커질 수 있음
- **낮게 설정**: 너무 빨리 손절될 수 있음
- **추천**: Target Profit과 비슷하거나 약간 크게

### Rebuy Threshold (재매수 하락폭)
- **높게 설정**: 재매수 기회가 적음 (안전)
- **낮게 설정**: 너무 빨리 재매수 (위험)
- **추천**: 1.5% ~ 2.5%

## ⚠️ 주의사항

### 1. 봇 실행 중 변경 가능
- 봇이 실행 중일 때도 설정 변경 가능
- 즉시 적용됨
- 기존 포지션에는 영향 없음 (새로운 거래부터 적용)

### 2. 최소 주문 금액
- 업비트: 최소 5,000 KRW
- Trade Amount를 5,000 미만으로 설정하지 마세요

### 3. 리스크 관리
- Target Profit과 Stop Loss를 너무 작게 설정하면 수수료 손실
- 업비트 수수료: 0.05% (매수) + 0.05% (매도) = 총 0.1%
- **최소 수익률**: 0.2% 이상 권장

### 4. 재매수 쿨다운
- Rebuy Threshold가 0%이면 바로 재매수 (위험)
